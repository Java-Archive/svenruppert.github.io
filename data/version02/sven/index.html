Big data collections with MapDB

2014-06-11 from Alexander Bischof

This article gives a short overview over the open source software MapDB which is now in version 1.0.3.

What is MapDB?

Original designed as a storage engine for an astronomical desktop application it had two design goals minimal overhead and simplicity. Over the time the engine had evolved and the third goal provide an alternative Java memory model was added. So now it is a storage engine which is specialized for big data collections and for that has some cool features. For example:h composite keys
bidirectiona
Compression
Faceting aka Histogram
Simulated Auto-Increment
Transactions (Note: a single transaction can only be used once)
Querying
